# SmartCart Coach 2.0 - Milestone 1

This is the first working milestone for the SmartCart Coach 2.0 capstone project.

Milestone 1 delivers:

- A React and TypeScript single page app
- A basic grocery list builder
- Local persistence with `localStorage`
- A simple client side Swap Score that uses price and calories
- Color coded visual feedback for the Swap Score

This milestone demonstrates the core Human Computer Interaction focus:
fast, clear, and supportive decision feedback for grocery items.

- Node.js 18 or later
- npm 9 or later
